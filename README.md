# Currency Converter

A simple currency converter powered by the [Currency Beacon API](https://currencybeacon.com/api-documentation).

## Currency Beacon API Key

This app required a Currency Beacon API key which can be obtained by creating an account at [Currency Beacon](https://currencybeacon.com/register).

Add your API key to `example.env`, then rename the file `.env`.

## Run Locally

`npm run dev`
